# Bilibili Web Clipper

专为 B站视频设计的浏览器扩展，实现将 B站内容转化为结构化知识的功能。

## 基本信息

- **作者**：haixiong1997
- **GitHub**：https://github.com/haixiong1997/Bilibili-Obsidian-Clipper
- **支持平台**：Chrome、Edge、Firefox
- **集成目标**：Obsidian 本地知识库

## 核心功能

### 1. 内容提取
- **标题提取**：自动获取视频标题和副标题
- **UP主信息**：提取作者ID、粉丝数、创作历史
- **字幕提取**：解析 B站特有的字幕格式和内容
- **视频简介**：提取视频描述、标签、分区信息
- **元数据**：BV号、CID、上传时间、播放量等

### 2. 格式转换
- **Markdown 生成**：将提取的内容转换为结构化的 Markdown 格式
- **图片处理**：自动下载封面图和内容截图到本地
- **链接保持**：保留 B站原始链接和引用

### 3. API 集成
- **Local REST API**：通过 API 直接写入 Obsidian 笔记库
- **目录配置**：支持自定义保存路径（如 `raw/03transcripts`）
- **批量处理**：支持连续处理多个视频内容

## 技术特点

### 解决 B站特有问题
- **页面结构差异**：B站采用不同播放器和页面架构，需要专用解析器
- **字幕格式特殊**：B站的 ASS/SRT 字幕格式需要专门处理
- **数据结构复杂**：B站特有的互动数据、弹幕、章节等需要单独处理

### 自动化程度
- **一键保存**：点击扩展图标即可完成整个流程
- **实时同步**：直接写入 Obsidian，无需手动导入
- **模板化输出**：统一的知识结构格式

## 使用场景

### 学习资料整理
- 将 B站课程视频转化为学习笔记
- 保留完整的知识脉络和关键内容
- 建立跨视频的知识关联

### 内容创作辅助
- 创作者的灵感收集和素材整理
- 参考资料的快速结构化
- 知识点的体系化整理

### 知识管理
- 观看视频时的实时笔记记录
- 视频内容的二次加工和提炼
- 个人知识库的丰富化

## 与现有工具的对比

| 特性 | Bilibili Web Clipper | 通用 Web Clipper |
|------|---------------------|------------------|
| B站支持 | ✅ 专用适配 | ❌ 无法处理 |
| 字幕提取 | ✅ 完整支持 | ❌ 有限支持 |
| 元数据 | ✅ B站完整数据 | ❌ 基础信息 |
| API 集成 | ✅ 专门适配 | ❌ 通用接口 |

## 集成方案

### 1. 独立使用
- 浏览器扩展 + Obsidian + Local REST API
- 适用于个人学习和知识整理

### 2. 批量处理
- 结合脚本进行大量视频的内容提取
- 适用于机构或创作者的批量处理需求

### 3. 工作流集成
- 嵌入到现有的 LLM Wiki 工作流中
- 作为多媒体摄入环节的核心工具
