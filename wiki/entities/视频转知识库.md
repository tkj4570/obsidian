# 视频转知识库

将 B站等平台的视频内容转化为本地 AI 可理解的知识库的工作流和方法。

## 核心概念

- **平台差异解决**：B站采用不同播放器与页面结构，通过专用工具（如 Bilibili Web Clipper）解决
- **字幕格式解析**：处理 B站特有的字幕格式和特殊数据结构
- **自动化工作流**：从视频收藏到知识生长的完整自动化流程

## 工具链

| 工具 | 功能 | 关联 |
|------|------|------|
| [[Obsidian]] | 本地知识库核心，支持双链与图谱可视化 | 本知识库本身 |
| [[Local REST API]] | 提供读写笔记库的标准化接口 | API 接口 |
| [[Bilibili Web Clipper]] | 专为 B站定制的抓取工具 | 浏览器扩展 |

## 工作流程

### 1. 环境准备
- 克隆 [[karpathy-llm-wiki-vault|Karpathy 知识库模板]] 到本地
- 在 Obsidian 中打开该仓库

### 2. 插件配置
- 安装 Local REST API 插件并启用 HTTP 服务器
- 安装 Bilibili Web Clipper 浏览器扩展

### 3. 视频入库
- 打开 B站学习视频
- 点击浏览器插件图标保存至 Obsidian
- 自动提取标题、UP主、字幕、简介，生成 markdown 文件

### 4. 知识处理
- 使用 Obsidian 功能自动完成双向链接与知识图谱构建
- 原文件自动归档，生成概念实体和摘要

## 技术特点

- **自动化程度高**：一键完成从视频到结构化知识的转换
- **保留完整元数据**：B 站特有的 BV 号、CID、上传时间等信息
- **双向链接自动构建**：符合 LLM Wiki 模式的知识网络要求
