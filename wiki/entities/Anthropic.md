# Anthropic

Claude 和 Claude Code 的开发商。其工程师 Boris Cherny 是 Loop Engineering 概念的重要倡导者。

## 相关产品

- **Claude**：大语言模型系列
- **[[Claude Code]]**：终端 AI 编程工具，内置 `/goal` 和 `/loop` 命令

## API 协议

Anthropic 使用自己的 Messages API 协议。DeepSeek 提供了兼容此协议的接口，使得 Claude Code 可以无缝切换到 DeepSeek 模型。
