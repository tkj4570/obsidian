# Headroom

Python 工具，在发送给 LLM 之前压缩输入（日志、工具输出、RAG 分块），减少 60-95% Token 消耗，保持回答质量。

- 仓库：https://github.com/chopratejas/headroom
- 部署方式：Python 库 / Agent / MCP Server
- 适用场景：RAG 系统、日志分析
