# Boris Cherny

Claude Code 之父，Anthropic 工程师。在 2026 年最新访谈中首次倡导 Loop Engineering 范式。

## 关键言论

> "我不再提示 Claude 了，我有一堆循环在运行，它们才是在提示 Claude 并判断接下来该做什么。我的工作变成了写循环。"

这句话被视为 Loop Engineering 概念的重要起源之一。
