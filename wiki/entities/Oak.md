# Oak

专为 AI Agent 设计的新型版本控制系统。虚拟挂载技术让 Agent 无需下载完整仓库即可工作，支持并行任务执行，减少 token 消耗。

- 仓库：https://oak.space/oak/oak
- 类比：类似 Google 的 google3 和微软 VFS for Git
- 状态：早期开发，缺少 Windows 支持
