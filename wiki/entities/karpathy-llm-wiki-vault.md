# karpathy-llm-wiki-vault

由社区基于 Andrej Karpathy 的 LLM Wiki 理念创建的开源知识库模板项目。

## 项目信息

- **GitHub**：https://github.com/jason-effi-lab/karpathy-llm-wiki-vault
- **类型**：Obsidian Vault 模板
- **用途**：提供 LLM Wiki 模式的开箱即用实现
- **社区**：基于 Karpathy 理念的二次开发实践

## 核心特性

### 1. 三层架构实现
- **Raw层**：结构化的原始资料目录
- **Wiki层**：自动生成的知识网络
- **Schema层**：配置化的规则和流程

### 2. 自动化工作流
- **智能摄入**：自动识别和分类新内容
- **链接生成**：自动构建双向链接网络
- **知识更新**：增量式的知识库维护

### 3. 插件集成
- **Local REST API**：支持外部工具集成
- **Dataview**：动态查询和展示
- **Templater**：模板化内容生成

## 使用场景

### 1. 个人知识管理
- 学习资料的系统性整理
- 研究论文的知识提取
- 个人想法的持续积累

### 2. 团队协作
- 团队知识库的共享和协作
- 项目文档的结构化管理
- 决策过程的记录和追踪

### 3. 教育培训
- 课程资料的体系化整理
- 学习路径的可视化
- 知识点的关联教学

## 与 MIP耀 工作的关联

MIP耀 的 B站视频转知识库工作流正是基于这个模板实现的：

1. **基础架构**：使用 karpathy-llm-wiki-vault 作为基础模板
2. **功能扩展**：添加多媒体内容处理能力
3. **工具集成**：集成 Bilibili Web Clipper 等专用工具
4. **工作流优化**：针对视频内容优化摄入流程

## 技术栈

| 组件 | 用途 | 说明 |
|------|------|------|
| Obsidian | 知识库核心 | 支持双链和图谱可视化 |
| Local REST API | API 接口 | 支持外部工具集成 |
| Bilibili Web Clipper | 内容摄入 | B站专用抓取工具 |
| Dataview | 数据查询 | 动态展示和统计 |
| Templater | 模板引擎 | 标准化内容生成 |
