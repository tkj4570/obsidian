# Vibe Coding 常见问题和解决

> 来源：程序员鱼皮，2026.06 | Vibe Coding 速查手册

## 核心观点

1. **Vibe Coding 的定义**：用自然语言描述需求，让 AI 写代码。类比——传统编程是"自己做饭"，Vibe Coding 是"点外卖"。

2. **AI 生成的代码能用但不完美**：可能有 bug、性能问题、安全隐患。需要测试、审查、优化，不能盲目信任。

3. **MVP 思维**：先做最简单能用的版本，再慢慢完善。这对于 Vibe Coding 尤其重要——避免一上来就陷入细节。

4. **工具选择分层**：零基础用零代码平台（Bolt.new、v0.dev），有基础用代码编辑器（Cursor、Claude Code）。零代码平台简单快速但灵活性有限。

5. **提示词质量决定输出质量**：具体、清晰、有结构——功能需求 + 界面要求 + 技术要求 + 参考案例。
