# 6月23日 github热榜：涨⭐前十项目全解析

> 来源：大雄AI实战，2026.06.23 | B站视频教程 | [视频链接](https://www.bilibili.com/video/BV15TjS6zEDF/)

## 核心观点

1. **AI工作流整合趋势**：前十项目中七个属于同一赛道，核心问题是"如何让AI不只是回答问题，而是能按流程做事"

2. **专业领域AI化**：视频制作、股票分析、网络安全等专业领域都在通过AI工具实现工作流程的自动化

3. **开发者关注焦点**：从agent skills和superpowers到agency agents，再到产品流程AI化，反映了AI应用层次的深化

4. **开源生态繁荣**：多项目二次上榜显示了开源AI工具的持续活跃和社区认可

## 热榜项目详情

### Top 1: OpenMontage
- **GitHub**: https://github.com/calesthio/OpenMontage
- **定位**: 开源智能体视频制作系统
- **亮点**：
  - 利用免费素材库制作真实动态视频（非图片动画）
  - 内置12条制作流水线，52个工具，500+智能体技能
  - 低成本案例（1.33美元制作皮克斯风格动画短片）
- **价值**: 将复杂视频制作流程完全自动化，降低创作门槛

### Top 2: Palmier Pro
- **GitHub**: https://github.com/palmier-io/palmier-pro
- **定位**: macOS原生AI视频编辑器
- **亮点**：
  - Swift开发，对标Premiere Pro体验
  - 内置生成式AI功能，支持在时间线内调用模型
  - 通过MCP协议连接CloudCode、Cursor等AI编程助手
- **价值**: 为Mac OS提供AI驱动的视频创作平台

### Top 3: Codebase Memory MCP
- **GitHub**: https://github.com/DeusData/codebase-memory-mcp
- **定位**: 高性能代码索引和知识图谱MCP服务器
- **亮点**：
  - 极快索引速度（几分钟处理Linux内核级代码库）
  - 支持158种编程语言，AST分析构建知识图谱
  - 单一静态二进制文件，零依赖
- **价值**: 解决AI编程助手在大型代码库中的效率痛点

### Top 4: Daily Stock Analysis
- **GitHub**: https://github.com/ZhuLinsen/daily_stock_analysis
- **定位**: AI驱动的多市场股票分析系统
- **亮点**：
  - 支持A股、港股、美股、日股、韩股等多市场
  - 内置15种量化策略，多轮追问的AI Agent功能
  - GitHub Actions实现零成本全自动定时分析
- **价值**: 开源免费的自动化股票分析与推送方案

### Top 5: Voicebox
- **GitHub**: https://github.com/jamiepine/voicebox
- **定位**: 本地运行的开源AI语音工作室
- **亮点**：
  - 支持7种TTS引擎和23种语言，零样本语音克隆
  - 全局语音输入热键，实时听写到任何文本框
  - 支持为MCP感知AI代理配置自定义语音输出
- **价值**: 整合语音克隆和输入功能，满足隐私需求

### Top 6: Anthropic Cybersecurity Skills
- **GitHub**: https://github.com/mukul975/Anthropic-Cybersecurity-Skills
- **定位**: AI智能体网络安全技能库
- **亮点**：
  - 817个结构化技能，覆盖29个安全领域
  - 兼容26+AI开发平台，即插即用
  - 遵循agentkills点IO开放标准
- **价值**: 解决AI智能体缺乏专业结构化安全知识的问题

### Top 7: GStack
- **GitHub**: https://github.com/garrytan/gstack
- **定位**: 将Claude Code转变为虚拟工程团队的工具集
- **亮点**：
  - 23个预设角色和8个核心工具
  - 所有工具均为Markdown格式的slash命令
  - 作者声称个人代码产出效率提升数百倍
- **价值**: 响应AI时代个人开发者高效工作的需求

### Top 8: Hermes Agent
- **GitHub**: https://github.com/NousResearch/hermes-agent
- **定位**: 自我学习、跨平台运行的AI智能体
- **亮点**：
  - 内置闭环学习系统，自主创建和优化技能
  - 支持Telegram、Discord、Slack等多平台接入
  - 可部署从$5 VPS到GPU集群的多种环境
- **价值**: 解决AI智能体缺乏长期记忆和技能进化的问题

### Top 9: AI Website Cloner Template
- **GitHub**: https://github.com/JCodesMore/ai-website-cloner-template
- **定位**: AI驱动的网站克隆模板生成器
- **亮点**：
  - 标准化AI智能体工作流，一键完成网站分析到重建
  - 支持多种主流AI编程智能体，推荐Claude Code
  - 生成基于Next.js 16、React、Tailwind CSS的现代项目
- **价值**: 将网站逆向工程自动化，降低前端开发门槛

### Top 10: Deer Flow
- **GitHub**: https://github.com/bytedance/deer-flow
- **定位**: 开源超级智能体框架
- **亮点**：
  - 协调子智能体记忆和沙箱执行复杂长时间任务
  - 内置沙箱环境提供安全隔离
  - 支持2.0版本重写，新增智能搜索等功能
- **价值**: 处理需要长时间运行的复杂任务流

## 赛道分析

### 主要赛道分类
1. **AI工作流工具**：OpenMontage、Palmier Pro、GStack、Hermes Agent、AI Website Cloner、Deer Flow
2. **代码库记忆**：Codebase Memory MCP
3. **技能库与规则集**：Anthropic Cybersecurity Skills
4. **专业领域应用**：Daily Stock Analysis、Voicebox

### 技术趋势
- **MCP协议普及**：多个项目支持Model Context Protocol
- **本地化运行**：强调隐私和离线能力的工具增多
- **模块化设计**：通过技能模块和子智能体实现复杂功能
- **跨平台支持**：支持从个人设备到云端集群的多种部署环境

### 开发者关注点演进
- **Agent Skills/Superpowers** → **Agency Agents** → **Product Process AI**
